Slider Test

Testing all slider functionality:

  • ✅ Auto-play (5 seconds)
  • ✅ Navigation buttons (< >)
  • ✅ Slide indicators (dots)
  • ✅ Keyboard controls (arrow keys)
  • ✅ Touch/swipe support
  • ✅ Hover to pause

How to Test:

  1. Auto-play: Wait 5 seconds to see slides change automatically
  2. Navigation: Click the < and > buttons
  3. Indicators: Click the dots at the bottom
  4. Keyboard: Use left/right arrow keys
  5. Touch: Swipe left/right on mobile
  6. Hover: Hover over slider to pause auto-play

Console Debugging:

Open browser developer tools (F12) and check the Console tab for detailed logging.
